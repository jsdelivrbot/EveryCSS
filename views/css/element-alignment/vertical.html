To vertically align one line of text to the middle of its container, set the line-height property to the height of the container. You then no longer need to specify the height of the container.

<div> This text is vertically aligned to the middle </div> div { line-height: 200px; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/t0rzayn5/1/'}) %>

If you have multiple lines of text, wrap it in an element with display: table-cell and set its container to display: table.

You can then use the vertical-align property on the text and set it to either top, middle or bottom.

<div> <span> This is some text that can span multiple lines and is also vertically centered inside its container. </span> </div> div { display: table; height: 200px; } span { display: table-cell; vertical-align: middle; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/cd2kxxg5/8/'}) %> <%- include(btu+'/templates/headingend.ejs') %> <%- include(btu+'/templates/heading.ejs', {'name': 'Images'}) %>

To vertically align one line of images, use the line-height property to specify the height of the image container, and then use the vertical-align property on the images to specify where they should be positioned.

<div> <img src="..."> </div> div { line-height: 200px; } img { vertical-align: middle; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/sqvvr4v0/5/'}) %>

The different values you can use are top, middle and bottom.

To vertically align block elements inside its container, give the container position: relative and then give the child position: absolute. What comes after that depends on whether you are centering the element, or aligning it to the bottom.

Vertically aligning to center:

To vertically align an element to the center of its container, set top: 50% for the child and then, since the anchor point of an element is at the top left, you must also shift the child upwards by half its height.

The original way of doing this was by applying a negative margin of half the child's height.




<%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/ju4fg6kx/9/'}) %>

However, a more modern and much more elegant solution (which doesn't require you to specify the height of the child) is to use the transform property to translate the child upwards.

<div class="container"> <div class="child"> Content </div> </div> .container { position: relative; height: 250px; } .child { position: absolute; top: 50%; transform: translateY(-50%); } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/ju4fg6kx/10/'}) %>

The final way of centering the child is to stretch it to all 4 edges of the container. By using margin: auto the child will be centered since its width and height are smaller than the container and won't be able to reach the edges.

.child { position: absolute; top: 0; bottom: 0; left: 0; right: 0; width: 100px; height: 100px; margin: auto; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/ju4fg6kx/13/'}) %>

Aligning to the bottom:

This is very simple to achieve. All you have to do is set bottom: 0px on the child element (or however far you want it from the bottom).

<div class="container"> <div class="child"> Content </div> </div> .container { position: relative; height: 250px; } .child { position: absolute; bottom: 0px; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/9m95muxa/4/'}) %> <%- include(btu+'/templates/headingend.ejs') %>
