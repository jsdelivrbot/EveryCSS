To horizontally align any inline element, use the text-align property on the parent element.

This includes inline, inline-block, inline-table, inline-flex, etc.

  • text-align: left - Aligns element to the left
  • text-align: center - Aligns element to the center
  • text-align: right - Aligns elements to the right

Example 1:

<p> This text is centered </p> p { text-align: center; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/71hf344y/'}) %>

Example 2:

<div> <div class="inline-block"> A </div> <div class="inline-block"> B </div> <div class="inline-block"> C </div> </div> div { text-align: center; } .inline-block { display: inline-block; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/gy5d5Lk7/4/'}) %> <%- include(btu+'/templates/headingend.ejs') %> <%- include(btu+'/templates/heading.ejs', {'name': 'Block elements'}) %>

If you have a display: block element and you have specified a width, you can center it by setting margin-left and margin-right to auto.

<div> This div will be aligned to the center. </div> div { width: 200px; margin: 0px auto; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/n19p6g40/2/'}) %>

To align a block element to the right, just set margin-left: auto.

<div> This div is aligned to the right. </div> div { width: 200px; margin-left: auto; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/pL48sgt9/3/'}) %>

If you don't specify the width of a block element you won't be able to center it since it would be full width. However, if you'd like the width to be automatically determined by the width of its contents, consider switching to a display: inline-block element and then centering it.

There are a few ways of centering fixed and absolutely positioned elements.

The first way is to set the left property to half of the parent element's width minus half of the element's width.

<div> This fixed div is centered. </div> div { position: fixed; width: 200px; top: 10px; left: calc(50% - 100px); } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/rye9cmpL/'}) %>

However, this isn't very useful if you don't know the width of the element. A simpler solution is to use the transform property.

<div> This fixed div is centered without specifying its width </div> div { position: fixed; top: 10px; left: 50%; transform: translateX(-50%); } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/jbu7mpaL/'}) %>

To align a fixed or absolutely positioned element to the right, simply use the right property to set how far from the right the element should be.

<div> This fixed div is aligned to the right </div> div { position: fixed; top: 20px; right: 20px; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/nfygkud0/'}) %> <%- include(btu+'/templates/headingend.ejs') %>
