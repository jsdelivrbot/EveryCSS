To horizontally align any inline element within its parent, use the text-align property on the parent element.

This includes elements displayed as inline, inline-block, inline-table, inline-flex and inline-grid

  • text-align: left - aligns element to the left
  • text-align: center - aligns element to the center
  • text-align: right - aligns elements to the right

Example 1:

<p> This text is centered </p> p { text-align: center; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/71hf344y/'}) %>

Example 2:

<div> <div class="inline-block"> A </div> <div class="inline-block"> B </div> <div class="inline-block"> C </div> </div> div { text-align: center; } .inline-block { display: inline-block; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/gy5d5Lk7/4/'}) %> <%- include(btu+'/templates/headingend.ejs') %> <%- include(btu+'/templates/heading.ejs', {'name': 'Block elements'}) %>

To horizontally align any block element that has a specified width within its parent, use the margin property.

This includes elements displayed as block, flex, grid and table

  • margin-right: auto - pushes element to the left of its available space
  • margin-left: auto and margin-right: auto - pushes element to the center of its available space
  • margin-left: auto - pushes element to the right of its available space

Example 1:

<div> This div is aligned to the center. </div> div { width: 200px; margin: 0px auto; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/n19p6g40/2/'}) %>

Example 2:

<div> This div is aligned to the right. </div> div { width: 200px; margin-left: auto; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/pL48sgt9/3/'}) %>

Example 3:

<div class="container"> <div> LEFT </div> <div> CENTER </div> <div> CENTER </div> </div> .container { display: flex; } .container > div:nth-child(1) { margin-right: auto; } .container > div:nth-child(3) { margin-right: auto; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/u6sgb503/3/'}) %>

To horizontally align elements positioned fixed and absolute within its parent or the window, use the following properties.

  • left: 0px - aligns element to the left
  • left: calc(50% - HALF_OF_ELEMENT_WIDTH) - aligns element to the center
  • left: 50% and transform: translateX(-50%) - aligns element to the center
  • right: 0px - aligns elements to the right

Example 1:

<div> This fixed div is centered. </div> div { position: fixed; width: 200px; top: 10px; left: calc(50% - 100px); } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/rye9cmpL/'}) %>

Example 2:

<div> This fixed div is centered without knowing its width. </div> div { position: fixed; top: 10px; left: 50%; transform: translateX(-50%); } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/jbu7mpaL/'}) %>

Example 3:

<div> This fixed div is aligned to the right. </div> div { position: fixed; top: 20px; right: 20px; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/nfygkud0/'}) %> <%- include(btu+'/templates/headingend.ejs') %> <%- include(btu+'/templates/heading.ejs', {'name': 'Flex items'}) %>

If you have a flex container with flex-direction: row (default) you can use the justify-content property to align its flex items horizontally.

All the property values are:

  • flex-start - items are positioned at the beginning of the container
  • flex-end - items are positioned at the end of the container
  • center - items are positioned at the center of the container
  • space-between - items are positioned with equal space between the lines
  • space-around - items are positioned with equal space around the lines

Example:

<div class="container"> <div> A </div> <div> B </div> </div> .container { display: flex; width: 100%; justify-content: center; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/kmr1rhz7/10/'}) %>

Additionally, you can push the flex items individually using the margin property to create a more complex layout. <%- include(btu+'/templates/jump.ejs', {'jump': 'Block elements', 'name': 'See details here.'}) %>



If you have a flex container with flex-direction: column you can use the align-items property to align its flex items horizontally.

All the property values are:

  • stretch (default) - items are stretched to fit the container
  • flex-start - items are positioned at the beginning (left) of the container
  • flex-end - items are positioned at the end (right) of the container
  • center - items are positioned at the center of the container
  • baseline - items are positioned at the baseline of the container

Example:

<div class="container"> <div> A </div> <div> B </div> </div> .container { display: flex; width: 100%; flex-direction: column; align-items: center; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/gx80whvm/'}) %>

To align items on the inline / row / horizontal axis use justify-items on the grid container or use justify-self on the individual grid items. Items will be aligned inside the total space given to them.

All the property values are:

  • stretch - the item is stretched to fill its available space
  • start or left - the item is positioned at the beginning of its available space
  • end or right - the item is positioned at the end of its available space
  • center - the item is positioned at the center of its available space

Example 1:

<div class="grid"> <div> Item 1 </div> <div> Item 2 </div> ... </div> .grid { display: grid; grid-template-columns: repeat(2, 1fr); justify-items: center; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/hzfrn1og/4/'}) %>

Example 2:

<div class="grid"> <div class="item1"> Item 1 </div> <div class="item2"> Item 2 </div> <div class="item3"> Item 3 </div> <div class="item4"> Item 4 </div> </div> .grid { display: grid; grid-template-columns: repeat(2, 1fr); } .item1 {justify-self: left;} .item2 {justify-self: right;} .item3 {justify-self: center;} .item4 {justify-self: stretch;} <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/0njhtebw/1/'}) %> <%- include(btu+'/templates/headingend.ejs') %> <%- include(btu+'/templates/heading.ejs', {'name': 'Floating'}) %>

The CSS float property floats an element to the left or to the right, allowing text to wrap around it.

All the property values are:

  • left - the element floats to the left of its container
  • right - the element floats to the right of its container

Example:

<img src="http://placehold.it/50x50"> This text will wrap around the image. img { float: left; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/e1854agc/12/'}) %> <%- include(btu+'/templates/headingend.ejs') %>

The CSS clear property is used on an element to specify what elements can float beside it. This property can be used to ensure an element doesn't wrap around floated elements, and instead, gets pushed underneath them. Note: This property only clears elements preceding it in the document structure and can only be applied to display: block elements.

All the property values are:

  • left - pushes the element under any float: left elements
  • right - pushes the element under any float: right elements
  • both - pushes the element underneath any floated elements

Example:

<img src="http://placehold.it/120x110"> This text will wrap around the image. <div class="clear"> This text isn't allowed to wrap around floated elements, and therefore is pushed underneath the image. </div> img { float: right; } .clear { clear: right; } <%- include(btu+'/templates/jsfiddlelink.ejs', {'link': 'https://jsfiddle.net/79gdnyt1/32/'}) %>
